# Event Management Frontend

Frontend application for managing events and venues at ආලකමන්දා Hotel.

## Features

- Event booking and management
- Venue management
- Dashboard with statistics
- Event listing and filtering
- Venue capacity management

## Tech Stack

- React 19
- Vite
- React Router
- Axios
- Lucide React Icons
- TanStack Query

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Backend Connection

The backend runs on port 8087. In development, the API is accessed at:
`http://localhost:8087/api/events`
